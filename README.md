# LIRI

Welcome to LIRI a text recognition app that allows you to:
1. View your tweets: node liri.js myTweets
2. Search music in spotify: node liri.js spotifyFind "NAME OF YOUR SONG"
3. Search movies: node liri.js movieFind "NAME OF YOUR MOVIE"
4. Do a preselected task: node liri.js do-what-it-says 

Run the text to the left on your command line and watch the magic occur!
